# ローカルonly対話MVP 実装計画

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** ローカル PC 上で「マイク入力 → VAD → STT → フロント LLM(Ollama) → 文分割 → TTS(GPT-SoVITS) → スピーカ再生」を低遅延でループさせ、barge-in 対応の対話 MVP を Discord 非依存で動かす。

**Architecture:** 既存フェーズ1の I/F 分離設計と中央 `Orchestrator`(asyncio)をそのまま再利用し、Discord 固有の受信(receiver)・再生(playback)・bot 配線だけをローカル I/O(`sounddevice`)へ差し替える。マイクは非 async の PortAudio コールバックスレッドから 16kHz/mono/float32 の 512 サンプルフレームを `Orchestrator.feed_audio(user_id, np.ndarray)` に渡し、再生は Player 契約互換の `LocalSpeaker` が barge-in 対応で行う。TTS は TTSサーバ を GPT-SoVITS `api_v2.py` の `/tts` クライアントへ置換する。

**Tech Stack:** Python 3.11+ / sounddevice(ローカル音声 I/O) / silero-vad(VAD) / faster-whisper large-v3-turbo(STT) / Ollama `/api/chat` ストリーミング(フロント LLM) / GPT-SoVITS `api_v2.py` `/tts`(TTS) / aiohttp / numpy / pytest + pytest-asyncio。

## Global Constraints

- **音声の内部表現**は **16000 Hz / mono / float32(振幅 [-1.0, 1.0] 正規化)** に統一する。フェーズ1から不変。
- **Player 契約(再生)**: `is_playing(self) -> bool`(同期)、`stop(self) -> None`(同期・barge-in 用)、`async play_and_wait(self, wav: bytes) -> bool`(自然終了=`True`、barge-in 中断=`False`)。`LocalSpeaker`(L3)と Orchestrator が呼ぶ Player はこの契約で完全一致する。
- **tts 契約(合成)**: `async (text: str) -> bytes`(完結した WAV バイト列を返す)。Orchestrator へは `functools.partial(synthesize, session=..., ref_audio_path=..., prompt_text=..., ...)` を `tts` として注入し、この契約を満たす(L2 / L5)。
- **feed_audio 契約(入力)**: `Orchestrator.feed_audio(user_id: int, audio: np.ndarray)` は **非 async のキャプチャスレッド**(PortAudio コールバック)から呼ばれる(内部で VAD ワーカースレッドへ submit し、確定イベントだけイベントループへ marshalling する)。`MicCapture`(L4)は `make_on_audio(orch)` 経由でこれを駆動する。
- **GPT-SoVITS TTS**: `POST {base_url}/tts` に JSON を送る。必須フィールドは `text` / `text_lang` / `ref_audio_path` / `prompt_lang`。`media_type="wav"`、`streaming_mode=False`。既定ポートは **9880**。出力は **自己記述 WAV(サンプルレートはモデル依存)** で、`LocalSpeaker` が `wave` でヘッダを読み取って再生する。サーバ側で重み(GPT/SoVITS)は起動時ロード済みの前提で、クライアントは重み管理を行わない。
- **コミット**: author / committer はユーザー自身の git 設定を使う。コミットメッセージ末尾には空行を1行はさんで次のトレーラを付ける:

  ```
  Co-Authored-By: Claude <noreply@anthropic.com>
  ```

- **テスト方針**: ユニットテストは **音声ハードウェア・外部サービス一切不要**。fake(`_FakeSession` / `FakeSd` / `_FakeOrch` 等)を注入して検証する。実機・実サービスを要するテストは `@pytest.mark.integration` で分離し、重い依存は各テスト内で `pytest.importorskip(...)` する。既定の単体実行は `-m "not integration"` で回す。
- **依存の段階導入**: `pyproject.toml` の `[project.optional-dependencies]` に `[local]` extra を追加し、`sounddevice` 依存をそこに置く(`all = ["talk-ai[voice,ml,local]"]` に含める)。`sounddevice` はバックエンド注入式モジュール(`speaker.py` / `mic.py`)が `__init__` 内で遅延 import するので、ユニットテストでは import されない。

    DEFAULT_TTS_TIMEOUT,

_FAKE_WAV = b"RIFF\x24\x00\x00\x00WAVEfake-pcm-bytes"


class _FakeResponse:
    def __init__(self, body: bytes, status: int = 200):
        self._body = body
        self.status = status
        self.raised = False

    async def __aenter__(self):
        return self

    async def __aexit__(self, *exc):
        return False

    def raise_for_status(self):
        self.raised = True
        if self.status >= 400:
            raise aiohttp.ClientResponseError(
                request_info=None, history=(), status=self.status
            )

    async def read(self) -> bytes:
        return self._body

    """aiohttp.ClientSession 互換の最小フェイク。post() は同期で async ctx を返す。"""

        self._status = status

import aiohttp

# config.tts_timeout_s = 15.0 と一致させる。
DEFAULT_TTS_TIMEOUT = aiohttp.ClientTimeout(total=15.0)


async def synthesize(
    text: str,
    *,
    session: aiohttp.ClientSession,
    ref_audio_path: str,
    prompt_text: str = "",
    text_lang: str = "ja",
    prompt_lang: str = "ja",
    speed_factor: float = 1.0,
    base_url: str = "http://localhost:9880",
    timeout: aiohttp.ClientTimeout = DEFAULT_TTS_TIMEOUT,
    extra: dict | None = None,
) -> bytes:
    """text を 1 リクエストで合成し、完結した WAV バイト列を返す。

    ref_audio_path は GPT-SoVITS サーバホスト上の参照音声へのパス。
    extra で top_k / temperature 等の追加パラメータを上書き/追加できる。
    """
    payload: dict = {
        "text": text,
        "text_lang": text_lang,
        "ref_audio_path": ref_audio_path,
        "prompt_text": prompt_text,
        "prompt_lang": prompt_lang,
        "speed_factor": speed_factor,
        "media_type": "wav",
        "streaming_mode": False,
    }
    if extra:
        payload.update(extra)

    url = base_url.rstrip("/") + "/tts"
    async with session.post(url, json=payload, timeout=timeout) as resp:
        resp.raise_for_status()
        return await resp.read()


async def synthesize_default(
    text: str,
    *,
    ref_audio_path: str,
    **kwargs,
) -> bytes:
    """自前で ClientSession を開いて synthesize に委譲する（統合/手動実行用）。

    アプリ本体は通常、共有 session を注入した synthesize を使うこと。
    """
    async with aiohttp.ClientSession() as session:
        return await synthesize(
            text,
            session=session,
            ref_audio_path=ref_audio_path,
            **kwargs,
        )

```python
import asyncio
import io
import wave
from typing import Optional, Tuple

import numpy as np


def _decode_wav(wav_bytes: bytes) -> Tuple[np.ndarray, int, int]:
    """WAV bytes -> (data(frames,channels) float32[-1,1], framerate, channels)。

    GPT-SoVITS の出力は 16-bit mono(sample rate はモデル依存, 通例 32000Hz)。
    16-bit 前提で int16 -> float32 へ正規化する。
    """
    with wave.open(io.BytesIO(wav_bytes), "rb") as wf:
        rate = wf.getframerate()
        channels = wf.getnchannels()
        width = wf.getsampwidth()
        nframes = wf.getnframes()
        raw = wf.readframes(nframes)
    if width != 2:
        raise ValueError(f"unsupported sample width: {width} (expected 16-bit)")
    i16 = np.frombuffer(raw, dtype="<i2").astype(np.float32) / 32768.0
    data = i16.reshape(-1, channels)
    return data, rate, channels


class LocalSpeaker:
    """ローカルスピーカ再生 + barge-in。Orchestrator の Player 契約互換。

    再生は注入された sounddevice 互換バックエンドの OutputStream(callback 方式)で行う。
    callback はチャンクを index 進行で書き込み、末尾 or 中断で sd.CallbackStop を送出。
    finished_callback はイベントループへ marshalling して完了を通知する。
    """

    def __init__(self, *, sd=None, loop=None):
        if sd is None:
            import sounddevice as sd  # 実機でのみ遅延 import(テストは fake を注入)
        self._sd = sd
        self._loop = loop
        self._interrupted = False
        self._stream = None

    def is_playing(self) -> bool:
        return self._stream is not None

    def stop(self) -> None:
        self._interrupted = True
        stream = self._stream
        if stream is not None:
            self._stream = None
            try:
                stream.abort()
            except Exception:
                pass
            try:
                stream.close()
            except Exception:
                pass

    async def play_and_wait(self, wav_bytes: bytes) -> bool:
        loop = self._loop or asyncio.get_running_loop()
        self._interrupted = False
        data, rate, channels = _decode_wav(wav_bytes)
        total = len(data)
        idx = 0
        done = asyncio.Event()

        def callback(outdata, frames, time_info, status):
            nonlocal idx
            if self._interrupted:
                raise self._sd.CallbackStop
            chunk = data[idx : idx + frames]
            n = len(chunk)
            outdata[:n] = chunk
            idx += n
            if n < frames:
                outdata[n:] = 0.0  # 最終(部分)バッファを 0 埋め
                raise self._sd.CallbackStop

        def finished_callback():
            loop.call_soon_threadsafe(done.set)

        stream = self._sd.OutputStream(
            samplerate=rate,
            channels=channels,
            dtype="float32",
            callback=callback,
            finished_callback=finished_callback,
        )
        self._stream = stream
        stream.start()
        try:
            await done.wait()
        finally:
            if self._stream is stream:  # stop() が先に閉じていなければ後始末
                self._stream = None
                try:
                    stream.close()
                except Exception:
                    pass
        return not self._interrupted
```

```python
from typing import Callable, Optional

import numpy as np

from talk_ai.config import SAMPLE_RATE_HZ, VAD_WINDOW_SAMPLES


class MicCapture:
    """ローカルマイクから 16kHz mono float32 を取り込み、512 サンプルのフレームを
    `on_audio(user_id, frame)` へ転送する。

    sounddevice の InputStream コールバックは PortAudio スレッド上で実行されるため、
    `on_audio` は非 async スレッドから呼ばれる前提(=> orch.feed_audio を直接駆動できる)。
    テストでは `sd` に fake モジュールを注入し、ハードウェア不要で検証する。
    """

    def __init__(
        self,
        on_audio: Callable[[int, np.ndarray], None],
        *,
        user_id: int = 0,
        samplerate: int = SAMPLE_RATE_HZ,
        blocksize: int = VAD_WINDOW_SAMPLES,
        device=None,
        sd=None,
    ) -> None:
        if sd is None:
            import sounddevice as sd  # 遅延 import(テストは fake を注入)
        self._sd = sd
        self._on_audio = on_audio
        self._user_id = user_id
        self._samplerate = samplerate
        self._blocksize = blocksize
        self._device = device
        self._stream = None

    def _cb(self, indata, frames, time, status) -> None:
        # indata: shape (frames, 1) float32 -> (frames,) float32
        frame = np.asarray(indata, dtype=np.float32).reshape(-1)
        self._on_audio(self._user_id, frame)

    def start(self) -> None:
        if self._stream is not None:
            return
        self._stream = self._sd.InputStream(
            samplerate=self._samplerate,
            channels=1,
            dtype="float32",
            blocksize=self._blocksize,
            device=self._device,
            callback=self._cb,
        )
        self._stream.start()

    def stop(self) -> None:
        if self._stream is None:
            return
        stream, self._stream = self._stream, None
        stream.stop()
        stream.close()

    def close(self) -> None:
        self.stop()

    def __enter__(self) -> "MicCapture":
        self.start()
        return self

    def __exit__(self, exc_type, exc, tb) -> None:
        self.stop()
```

import aiohttp


async def _ollama_ok(session, base_url: str) -> bool:
    """Ollama: GET /api/tags が 200 を返せば稼働とみなす。"""
    try:
        async with session.get(f"{base_url}/api/tags") as resp:
            return resp.status == 200
    except aiohttp.ClientError:
        return False


async def _reachable(session, base_url: str) -> bool:
    """GPT-SoVITS: 専用ヘルスエンドポイントが無い。

    何らかの HTTP 応答(404/400 含む)が返れば到達可能とみなし、
    接続エラー(ClientError)のみ down とする。
    """
    try:
        async with session.get(f"{base_url}/") as resp:
            _ = resp.status   # 応答が返れば到達可能
            return True
    except aiohttp.ClientError:
        return False


async def check_local_services(session, *, ollama_url: str, gptsovits_url: str) -> dict:
    """ローカル構成(Ollama + GPT-SoVITS)の疎通を確認する。"""
    return {
        "ollama": await _ollama_ok(session, ollama_url),
        "gptsovits": await _reachable(session, gptsovits_url),
    }

```python
import asyncio
import functools

import aiohttp

from talk_ai.config import Config
from talk_ai.health import check_local_services
from talk_ai.llm.front_client import stream_chat
from talk_ai.orchestrator import Orchestrator, make_on_audio
from talk_ai.voice.mic import MicCapture
from talk_ai.voice.speaker import LocalSpeaker
from talk_ai.voice.stt import build_whisper
from talk_ai.voice.tts_gptsovits import synthesize
from talk_ai.voice.vad import SileroVad


def build_orchestrator(
    config,
    *,
    session,
    loop,
    transcriber=None,
    player=None,
    vad_factory=SileroVad,
):
    """設定と長命セッションから Orchestrator を結線する(単体テスト可能)。

    transcriber / player を注入しなければ、それぞれ実装(whisper / LocalSpeaker)を生成する。
    """
    if transcriber is None:
        transcriber = build_whisper(
            config.whisper_model,
            device=config.whisper_device,
            compute_type=config.whisper_compute_type,
        )
    if player is None:
        player = LocalSpeaker(loop=loop)

    tts = functools.partial(
        synthesize,
        session=session,
        base_url=config.gptsovits_url,
        ref_audio_path=config.gptsovits_ref_audio_path,
        prompt_text=config.gptsovits_prompt_text,
        text_lang=config.gptsovits_text_lang,
        prompt_lang=config.gptsovits_prompt_lang,
        speed_factor=config.gptsovits_speed_factor,
    )
    llm_stream = functools.partial(
        stream_chat,
        base_url=config.ollama_url,
        session=session,
    )

    # フェーズ1 bot.py と同一の明示 kwargs 配線。Orchestrator の署名は変更しない。
    return Orchestrator(
        transcriber=transcriber,
        llm_stream=llm_stream,
        tts=tts,
        player=player,
        model=config.ollama_model,
        vad_factory=vad_factory,
        history_max_turns=config.history_max_turns,
        vad_threshold=config.vad_threshold,
        vad_silence_ms=config.vad_silence_ms,
        bargein_trigger_ms=config.bargein_trigger_ms,
        fallback_text=config.fallback_text,
        stt_timeout=config.stt_timeout_s,
        tts_timeout=config.tts_timeout_s,
        play_timeout=config.play_timeout_s,
        loop=loop,
    )


async def run_local(config: Config) -> None:
    """ローカル(マイク+スピーカ)で会話ループを常駐させる。integration 専用。"""
    loop = asyncio.get_running_loop()
    timeout = aiohttp.ClientTimeout(total=None, sock_read=300)
    async with aiohttp.ClientSession(timeout=timeout) as session:
        status = await check_local_services(
            session,
            ollama_url=config.ollama_url,
            gptsovits_url=config.gptsovits_url,
        )
        for name, ok in status.items():
            print(f"[health] {name}: {'OK' if ok else 'DOWN'}")
        if not all(status.values()):
            raise RuntimeError(f"必要なサービスに接続できません: {status}")

        orch = build_orchestrator(config, session=session, loop=loop)
        mic = MicCapture(
            make_on_audio(orch),
            user_id=config.local_user_id,
            device=config.input_device,
        )
        mic.start()
        print("マイク入力を開始しました。Ctrl+C で終了します。")
        try:
            await asyncio.Event().wait()   # KeyboardInterrupt まで常駐
        finally:
            mic.stop()


def main() -> None:
    try:
        asyncio.run(run_local(Config()))
    except KeyboardInterrupt:
        print("\n終了します。")


if __name__ == "__main__":
    main()
```

型整合性の自己レビュー。本計画の主要な3契約 + プレースホルダ不在を確認した。

- **Player 契約(L3 ⇔ Orchestrator)**: 完全一致。`LocalSpeaker`(L3)は `is_playing(self) -> bool`(同期)、`stop(self) -> None`(同期)、`async play_and_wait(self, wav_bytes: bytes) -> bool`(自然終了=`True` / barge-in 中断=`False`)を実装し、Orchestrator(Task 11/12)が呼ぶ Player 契約と署名・戻り値の意味が一致する。L5 では `player=None` 既定で `LocalSpeaker(loop=loop)` を生成し、`loop` は `LocalSpeaker.__init__(self, *, sd=None, loop=None)` が受理する。
- **tts 契約(L2 ⇔ Orchestrator 注入 in L5)**: 完全一致。`async (text: str) -> bytes`。L2 の `synthesize(text, *, session, ref_audio_path, prompt_text, text_lang, prompt_lang, speed_factor, base_url, timeout, extra)` を、L5 が `functools.partial` で `session` / `base_url` / `ref_audio_path` / `prompt_text` / `text_lang` / `prompt_lang` / `speed_factor` を束縛し、残る位置引数 `text` のみで `await tts(text) -> bytes` を満たす。`DEFAULT_TTS_TIMEOUT.total == 15.0` は `config.tts_timeout_s` と一致。`tests/test_local_app.py` が partial の `func is synthesize` と束縛 kwargs を検証する。
- **feed_audio 署名(L4 mic ⇔ Orchestrator)**: 一致。`MicCapture._cb` は `indata (frames,1) float32` を `(frames,)` へ flatten し `on_audio(user_id, frame: np.ndarray)` を呼ぶ。`make_on_audio(orch)`(Task 11)経由で `Orchestrator.feed_audio(user_id: int, audio: np.ndarray)`(Task 12)を駆動する。呼び出しは非 async の PortAudio コールバックスレッドから行われ、Orchestrator 側で VAD ワーカースレッドへ submit する前提と整合。
